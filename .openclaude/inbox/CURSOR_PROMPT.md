# Paste this into your OpenClaude session (~/cit)

```
Read CLAUDE.md and .openclaude/inbox/TASK.md.

You are paired with Cursor on CIT:
- You own src/analysis/ and data collectors
- Cursor owns src/output/ and ./cit CLI

Complete the task in TASK.md, run the verification command, then append a short summary to .openclaude/inbox/DONE.md.
```

After you run that, OpenClaude and Cursor stay in sync via `TASK.md` / `DONE.md` and `./cit delegate "..."`.
